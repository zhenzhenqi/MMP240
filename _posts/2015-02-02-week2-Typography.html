Animate.css

  1. Let's first have some fun. Go to the link below and play with the different pre-canned animation effects here. http://daneden.github.io/animate.css/
  2. Let's have more fun by adding this htmp example code into our own html elements. You can swap "bounced" with anything from this list here. https://github.com/daneden/animate.css
  3. Now, let's add some jQuery to controll the timing of these effects. First, when index.html is fully loaded, fade in our navbar. {% highlight js %} $(document).ready(function () { $("nav").addClass('animated fadeIn'); }); {% endhighlight %}
  4. Now, let's tell jQuery to help us measure the location of a few things:
    how much is the entire document scrolled down
    our animated element's location from the top of the document. It's called offset.
    viewport height
  5. We will be able to tell wheather our element has entered into viewport by using this if statment

    offset - document_scoll_distance < than viewport height

    {% highlight js %} $(document).ready(function () { $("nav").addClass('animated fadeIn'); $("#featured_1").addClass('animated fadeInRight'); }); $(window).scroll(function(){ // console.log("scrolled!"); var how_much_document_scrolled = $(document).scrollTop(); // console.log(how_much_document_scrolled); var h1_offset_top = $('#tagline h1').offset().top; console.log(h1_offset_top); // console.log(how_much_document_scrolled + "," + h1_offset_top); if(h1_offset_top - how_much_document_scrolled < window.innerHeight){ $('#tagline h1').addClass('animated fadeInDown'); } var h3_offset_top = $('#founder h3').offset().top; if(h3_offset_top - how_much_document_scrolled < window.innerHeight-100){ $('#founder h3').addClass('animated lightSpeedIn'); } }); {% endhighlight %}
